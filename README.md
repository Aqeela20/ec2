Developed a cloud-based monitoring and traffic management solution using AWS services to ensure high availability, performance optimization, and fault tolerance of a web server deployed on Amazon EC2
Launched and configured an Amazon EC2 instance to host a web server.
Implemented CPU utilization monitoring using Amazon CloudWatch.
Configured a CloudWatch Alarm to trigger when CPU usage exceeds 80% threshold.
Integrated Amazon SNS to send real-time email notifications during high CPU usage.
Applied traffic management by restricting access to a maximum of 1000 concurrent users to prevent server overload.
Ensured system reliability and availability during peak traffic conditions.
